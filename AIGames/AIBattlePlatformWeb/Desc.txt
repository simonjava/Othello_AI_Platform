<h2>说明</h2>
<h3>系统功能：</h3>目前用户能在本系统上提交黑白棋AI代码，系统编译正确后，用户能和其他用户的AI代码进行比赛,系统尚未完全完善，会在近期完善，有建议或者bug欢迎发送邮件到chengsm@mail.ustc.edu.cn
<br/>
<h3>黑白棋规则：</h3>如果您不是很清楚规则，可参考百度百科中一个说明<a href="http://baike.baidu.com/view/24242.htm" target = "_blank">点击我</a>。
<br/>
<h3>接口说明：</h3>程序中用户只需接受输入一个棋盘(用scanf接受输入),然后输出你的落点位置即可(用printf输出，只需给出一步，系统会自动不停调用你和你的对手的程序)。具体可以参考<a href="example.txt" target="_blank">示例程序</a>
<br/>
目前轮到谁下，是否比赛已结束，系统都会自动判断,你无需写相关逻辑。若给你输入一个棋盘局面后，你的落点位置不合法，或者超时，超内存，都会判定你输，比赛自动结束~
<br/>
8x8的棋盘,1代表你的子，2代表对手的子，0代表空棋盘,棋盘坐标从0开始(0-7)
<br/>
<h3>输入输出</h3>
输入<br/>
0 0 0 0 0 0 0 0<br/>
0 0 0 0 0 0 0 0<br/>

0 0 0 0 0 0 0 0<br/>
0 0 0 0 0 0 0 0<br/>
0 0 0 0 0 0 0 0<br/>
输出<br/>
3 5<br/>
